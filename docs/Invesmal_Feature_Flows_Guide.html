2. Module 3.1 — User Management

Users register karte hain, login karte hain, profile update karte hain.

  1. Public URL: /register — name, email, password, role select (Founder / Investor / Mentor)
  2. System user create karta hai → auto login → /dashboard
  3. Profile dekhne ke liye: sidebar → profile link ya /users/{id}/profile
  4. Profile edit: /users/{id}/edit — avatar, bio, university
  5. Admin user verify karta hai: /admin/verification → Approve user
  6. Logout: POST /logout

Demo records: 10 users seeded

3. Module 3.2 — Startup Management

  1. Founder login → /startups (My Startups)
  2. Create: /startups/create — name, description, stage (idea/mvp/funded), industry, funding goal, logo
  3. Save → public page /startups/{id}
  4. Progress score aur funding % automatically calculate hota hai
  5. Public discover: /discover — search, filter stage/industry
  6. Admin verify: /admin/verification → Approve startup

Demo records: 12+ startups

4. Module 3.3 — Investment & Opportunity

  1. Investor /discover se startup dekhta hai
  2. Invest click → /investments/create — amount + message
  3. Status = pending → event InvestmentRequested
  4. Founder /investments par requests dekhta hai
  5. Founder /investments/{id} → Approve ya Reject
  6. Investor ko notification + email (preferences on hon to)

Demo records: 12+ investments (pending/approved/rejected mix)

5. Module 3.4 — Document Management

  1. Founder → /documents
  2. Upload form: name, type (pitch deck / business plan), optional startup, file
  3. Same startup + type par dubara upload → version 2, 3… auto increment
  4. Download: role-based (owner, founder, verified startup for investors)
  5. Delete: owner ya admin

Demo records: 12+ documents with versions

6. Module 3.5 — Communication & Meetings

Messages

  1. /conversations — list
  2. New chat: participant select → conversation create
  3. /conversations/{id} — messages; har 5 sec polling se naye messages
  4. Message send → MessageSent event → notifications + sentiment job

Meetings

  1. /meetings/create — invitee, title, date/time, location, notes
  2. Save → invitee ko notification
  3. Invitee /meetings/{id} → Accept / Decline

Demo records: 12+ meetings, 12+ conversations, 24+ messages

7. Module 3.6 — Administration

  1. Admin login → /admin/verification — pending users & startups
  2. /admin/moderation — flag / unflag startups
  3. /admin/monitoring — platform stats
  4. /admin/activity-logs — audit trail (12+ logs seeded)

8. Module 3.7 — Notifications

  1. In-app: bell icon → /notifications
  2. Events: investment, meeting, message, startup verified → auto create
  3. Email: SendEmailNotification listener (mail configured hon to)
  4. Preferences: gear icon → /notification-preferences — email/in-app toggles

Demo records: 12+ notifications

9. Module 3.8 — Reporting & Dashboard

  1. Admin → /reports — full platform stats
  2. Investor → /reports — investor activity summary
  3. Founder → /reports — startup performance
  4. Export CSV: /reports/export-csv
  5. Export report HTML (print to PDF): /reports/export-pdf

10. Module 3.9 — AI Analytics & Insights

  1. Login (any role except guest) → /ai/insights
  2. Dekho: engagement ranking, industry trends, competitive comparison (2 startups same industry)
  3. AI recommendations (OpenAI key ho to smart; warna default tips)

11. Module 3.10 — Sentiment Analysis

  1. /ai/sentiment — apni conversations list + sentiment badge
  2. Conversation select → /ai/sentiment/{id}
  3. Score, label (Positive/Neutral/Negative), breakdown %
  4. Refresh analysis: link with ?refresh=1
  5. Background: ProcessSentimentJob on new messages

Demo records: conversations mein pre-filled sentiment

12. Module 3.11 — AI Pitch Assistance

  1. Founder → /pitch-decks
  2. Generate: /pitch-decks/generate — title + startup description → AI deck
  3. Edit sections → Save → status final
  4. Analyze: /pitch-decks/{id}/analyze → /pitch-decks/{id}/analysis
  5. Upload PDF: /pitch-decks/upload
  6. Startup form mein pitch deck attach → public startup profile par dikhe

Demo records: 12+ pitch decks (draft/generated/analyzed/final)

  1. Step 1: Public landing / → search startup
  2. Step 2: Investor login → Discover → investment request
  3. Step 3: Founder login → Investments → approve
  4. Step 4: Meeting schedule + accept
